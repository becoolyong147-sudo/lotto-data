4. Generate new token (classic)
5. Note 随便填，Expiration 建议选 No expiration
6. 勾选 gist（只需要这一个权限）
7. 点 Generate token → 复制那串 ghp_xxx...（只显示一次！）

🧪 V27 策略实验室 — 按你的方法找候选号码

这是按你描述的六步法做的引导器。每一步系统会主动算给你看，你确认或调整后继续。
走完六步会输出候选号码 + 二次筛选。可保存为命名策略 → 跑历史回测看长期命中率。
六步算法本质：①连续龙长 → ②龙的历史分布对比 → ③形态频率 → ④后3D 龙长验证 → ⑤4D 点数段频率 → ⑥AND 交集

1第一步：维度扫描 — 找最强信号（看连续龙）

数根分组：G1 = 1,4,7 G2 = 2,5,8 G3 = 3,6,9　数根 0 = 爆冷不计
这里看的是「连续延续 / 断了」，不是「出现次数」。「当前龙=N」意思是从最新一期往前数，连续 N 期这个 G 至少出现一次，第 N+1 期断掉。
用法：龙越长 → 这个 G 在该维度上的「惯性」越强 → 第二步评估它会续龙还是断了。如果你想看「次数热度」请去数根透视看走势图。

2第二步：评估台柱 — 续龙还是放手
